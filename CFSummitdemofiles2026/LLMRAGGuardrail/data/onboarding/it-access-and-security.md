# IT Access and Security

## Account activation

New hires receive an account activation email from the IT Service Desk. The activation link expires after 24 hours. During activation, employees must create a password, enroll in multi-factor authentication, and confirm recovery contact information.

Employees must use the company password manager for company credentials. Reusing personal passwords for company systems is not permitted.

## Device setup

Company laptops must be enrolled in device management before accessing internal systems. Enrollment confirms disk encryption, endpoint protection, operating system patch level, and remote wipe capability.

If enrollment fails, employees should open an IT ticket with the device serial number and a screenshot of the error. Employees should not disable endpoint protection or install unapproved system extensions to work around setup issues.

## VPN and internal access

VPN access is required for internal admin tools, source repositories, and sensitive dashboards when working outside a company office. Employees should test VPN during the first two business days. If VPN fails, confirm MFA is active, restart the VPN client, and open an IT ticket if the problem continues.

Access requests must include the business reason, manager approval, and the system or repository name. Access is reviewed quarterly.

## Approved tools

Approved collaboration tools include company email, calendar, chat, video meetings, ticketing, source control, document storage, and the learning platform. Company files must not be stored in personal cloud drives, personal email, or consumer messaging apps.

Software not listed in the approved catalog requires security review before installation.

## Reporting incidents

Employees must report suspected phishing, lost devices, exposed credentials, or accidental sharing of company data immediately. Use the security hotline or the Report Security Incident button in the help desk portal. Fast reporting is expected; employees are not penalized for good-faith reports.
