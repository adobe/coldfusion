Simple RAG · chat()

simpleRAG().chat()

Multi-turn conversation that keeps the turn history so follow-ups stay on topic.

What this panel does

Same simpleRAG service, different entry point. .chat() retains prior messages in the session, so the model can resolve pronouns and follow-up questions (“what about its storage limit?”) against the previous turn.

The code

rag = simpleRAG(
    expandPath("../Documents/company-handbook.txt"),
    chatModel,
    { vectorStore: vs, chunkSize: 500, chunkOverlap: 100 }
);
rag.ingest();

// first turn
rag.chat("What is the annual leave entitlement?");

// follow-up — the service still remembers turn 1
rag.chat("How many days can I carry forward?");

When to use each

MethodUse for
ask()Standalone questions where each call is independent.
chat()Conversational flows where context from earlier turns matters.
