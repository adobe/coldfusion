Simple RAG · ask()

simpleRAG().ask()

One-shot, stateless Q&A grounded in a single document.

What this panel does

The app builds a simpleRAG service pointing at product-docs.txt, ingests it into Milvus, and exposes .ask(). Each question is answered independently — no memory of previous turns.

The code

chatModel = chatModel({
    provider  : "openai",
    modelName : "gpt-4o-mini",
    apiKey    : application.openaiKey
});

rag = simpleRAG(
    expandPath("../Documents/product-docs.txt"),
    chatModel,
    { vectorStore: vs, chunkSize: 500, chunkOverlap: 100 }
);
rag.ingest();

answer = rag.ask("What are the pricing plans?");

Key parameters

ParameterUsed for
sourceFile, folder or URL to ingest.
chunkSizeCharacters per chunk written to the vector store.
chunkOverlapCharacters shared between neighbouring chunks so sentences aren’t cut.
maxResultsTop-k chunks fetched per question.
minScoreMinimum similarity score a chunk must pass to be used.
